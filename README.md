Coursera: *Introduction to Web Development Capstone Project*

This project is a personal portfolio website developed as the final capstone for the Introduction to Web Development course. It showcases fundamental skills in semantic web structure, responsive design, and client-side interactivity.

**Features:**
HTML Structure: 
 Built using semantic HTML5 elements including header, nav, main, section, and footer.
  
**Accessibility:**
Includes lang="en" attributes and descriptive alt tags for all social media icons to support screen readers.

**CSS Styling:**
Features a custom design with a fixed navigation sidebar and distinct hover states.

**JavaScript Interactivity:**
 - Dynamic Date: Automatically updates the copyright year in the footer.
 - Read More Toggle: A functional "Read More" button that expands text using DOM manipulation.
 - Responsiveness: Utilizes CSS Media Queries (@media) to ensure the layout stacks vertically on mobile devices, preventing content overlap.
 - Navigation: Implemented internal anchor links (#) for smooth navigation between the "About" and "Contact" sections.
